# 👩‍⚕️ روانشناس هوش مصنوعی (AI Psychologist)

این پروژه یک **چت‌بات روانشناس حرفه‌ای** است که با استفاده از API مدل‌های OpenAI ساخته شده.  
کاربر می‌تواند به صورت آنلاین سوالات یا مشکلات خود را مطرح کند و ربات با لحن همدلانه و علمی پاسخ می‌دهد.  

---

## ✨ ویژگی‌ها
- رابط کاربری شبیه پیام‌رسان (چت حبابی)  
- پاسخ‌های هوشمند با لحن روانشناسی حرفه‌ای  
- ذخیره تاریخچه گفتگو در مرورگر (localStorage)  
- نمایش "در حال تایپ کردن…" برای حس طبیعی‌تر  
- قابل اجرا روی **Glitch** یا **Replit** بدون نیاز به سرور اختصاصی
